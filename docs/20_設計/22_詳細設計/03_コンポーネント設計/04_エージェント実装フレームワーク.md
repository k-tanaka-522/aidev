# aiDev エージェント実装フレームワーク

## 1. 概要

本ドキュメントでは、aiDevシステムにおけるAIエージェントの実装に関する共通フレームワークを定義します。このフレームワークに従うことで、各エージェントの実装の一貫性と相互運用性を確保し、将来の拡張性を高めます。

## 2. エージェント実装アーキテクチャ

### 2.1 全体アーキテクチャ

各エージェントは、以下のコンポーネントから構成される標準アーキテクチャを採用します：

```
 ┌─────────────────────────────────────────────────┐
 │              Agentコンテナ                      │
 │  ┌───────────────┐     ┌───────────────────┐   │
 │  │ AWS Lambda    │     │  AWS Bedrock      │   │
 │  │ ┌───────────┐ │     │ ┌───────────────┐ │   │
 │  │ │Agent      │ │━━━━▶│ │LLM (Claude)   │ │   │
 │  │ │Controller │ │◀━━━━│ │               │ │   │
 │  │ └───────────┘ │     │ └───────────────┘ │   │
 │  └───────┳───────┘     └───────────────────┘   │
 │          ┃                                     │
 │  ┌───────▼───────┐     ┌───────────────────┐   │
 │  │ System Prompt │     │ Amazon DynamoDB   │   │
 │  │ Generator     │     │ ┌───────────────┐ │   │
 │  │ ┌───────────┐ │     │ │Context Store  │ │   │
 │  │ │Templates  │ │     │ │               │ │   │
 │  │ └───────────┘ │     │ └───────────────┘ │   │
 │  └───────────────┘     └───────────────────┘   │
 └─────────────────────────────────────────────────┘
```

### 2.2 コンポーネント定義

1. **Agent Controller**
   - エージェントのメイン制御ロジック
   - リクエスト処理、状態管理、応答生成の調整
   - 他エージェントとの通信制御

2. **System Prompt Generator**
   - エージェント特性に基づくプロンプトテンプレート管理
   - コンテキストに応じたダイナミックプロンプト生成
   - エージェント指示の最適化

3. **LLM Interface**
   - AWS Bedrockとの連携
   - モデルパラメータ最適化
   - 応答の後処理

4. **Context Store**
   - セッション状態の保存と取得
   - 会話履歴の管理
   - エージェント間で共有されるコンテキストの管理

```javascript
class AgentController {
  constructor(agentConfig, contextStore) {
    this.agentId = agentConfig.agentId;

    this.contextStore = contextStore;
    this.promptGenerator = new SystemPromptGenerator(agentConfig);
    this.llmInterface = new LLMInterface(agentConfig.modelSettings);

  // メインハンドラー
  async processRequest(event) {
    try {
      // リクエスト検証
      const validatedRequest = this.validateRequest(event);
      
      // コンテキスト取得
      const context = await this.loadContext(validatedRequest);
      
      // プロンプト生成
```

### 4.2 状態管理

エージェントの状態は以下の遷移に従います：

```
 ┌──────────┐     ┌──────────┐     ┌──────────┐
 │  WAITING │────▶│PROCESSING│────▶│ RESPONSE │
 └────┬─────┘     └──────────┘     └────┬─────┘
      │                                  │
      │                                  │
      │                                  ▼
      │                            ┌──────────┐
      └───────────────────────────┤  HANDOFF  │
                                  └──────────┘
```

状態遷移ルール：
- WAITING: 入力待ち状態
- PROCESSING: リクエスト処理中、LLM生成中
- RESPONSE: 応答生成完了、ユーザーへの返答
- HANDOFF: 他エージェントへの引継ぎが発生

### 5.2 動的プロンプト生成

テンプレートベースの動的プロンプト生成ロジック：

```javascript
class SystemPromptGenerator {
  constructor(agentConfig) {
    this.baseTemplate = agentConfig.promptTemplate;
    this.dynamicSections = agentConfig.dynamicSections || {};
    this.agentName = agentConfig.name;
    this.agentId = agentConfig.agentId;
  }
  
  generatePrompt(request, context) {
    // テンプレート取得
    let prompt = this.baseTemplate;
    
    // 基本変数置換
    prompt = prompt.replace('{{agent_name}}', this.agentName)
                   .replace('{{agent_id}}', this.agentId);
    
    // コンテキスト要約生成
    const contextSummary = this.generateContextSummary(context);
    prompt = prompt.replace('{{context_summary}}', contextSummary);
    
    // 会話履歴フォーマット
    const conversationHistory = this.formatConversationHistory(
      context.conversationHistory || []
    );
    prompt = prompt.replace('{{conversation_history}}', conversationHistory);
    
    // ユーザー入力
    prompt = prompt.replace('{{user_message}}', request.userMessage || '');
    
    // タスク特化指示
    const specificTask = this.generateSpecificTask(request, context);
    prompt = prompt.replace('{{specific_task}}', specificTask);
    
    // 動的セクションの処理
    for (const [key, generator] of Object.entries(this.dynamicSections)) {
      const placeholder = `{{${key}}}`;
      if (prompt.includes(placeholder)) {
        const content = generator(request, context);
        prompt = prompt.replace(placeholder, content);
      }
    }
    
    return prompt;
  }
  
  // その他のヘルパーメソッド
  generateContextSummary(context) { /* 実装 */ }
  formatConversationHistory(history) { /* 実装 */ }
  generateSpecificTask(request, context) { /* 実装 */ }
}
```

## 6. LLM Interface 設計

### 6.1 AWS Bedrock 連携

AWS Bedrockとのインターフェースクラス：

```javascript
const { BedrockRuntimeClient, InvokeModelCommand } = require('@aws-sdk/client-bedrock-runtime');

class LLMInterface {
  constructor(modelSettings) {
    this.client = new BedrockRuntimeClient({ region: process.env.AWS_REGION });
    this.modelId = modelSettings.modelId || 'anthropic.claude-3-sonnet-20240229-v1:0';
    this.defaultParams = {
      maxTokens: modelSettings.maxTokens || 4096,
      temperature: modelSettings.temperature || 0.7,
      topP: modelSettings.topP || 0.9,
      stopSequences: modelSettings.stopSequences || []
    };
  }
  
  async generateResponse(prompt, overrideParams = {}) {
    try {
      // パラメータマージ
      const params = { ...this.defaultParams, ...overrideParams };
      
      // リクエスト構築
      const request = {
        modelId: this.modelId,
        contentType: 'application/json',
        accept: 'application/json',
        body: JSON.stringify({
          anthropic_version: 'bedrock-2023-05-31',
          max_tokens: params.maxTokens,
          temperature: params.temperature,
          top_p: params.topP,
          stop_sequences: params.stopSequences,
          messages: [
            {
              role: 'user',
              content: prompt
            }
          ]
        })
      };
      
      // モデル呼び出し
      const command = new InvokeModelCommand(request);
      const response = await this.client.send(command);
      
      // レスポンス解析
      const responseBody = JSON.parse(new TextDecoder().decode(response.body));
      
      return {
        content: responseBody.content[0].text,
        usage: responseBody.usage,
        modelId: this.modelId
      };
    } catch (error) {
      console.error('LLM API Error:', error);
      throw new LLMServiceError(error.message);
    }
  }
}

class LLMServiceError extends Error {
  constructor(message) {
    super(message);
    this.name = 'LLMServiceError';
  }
}
```

### 7.2 コンテキスト操作メソッド

```javascript
class ContextStore {
  constructor(sessionId) {
    this.sessionId = sessionId;
    this.dynamoDb = new DynamoDBClient({ region: process.env.AWS_REGION });
    this.docClient = DynamoDBDocumentClient.from(this.dynamoDb);
  }
  
  // セッションコンテキスト取得
  async getSessionContext() {
    const params = {
      TableName: process.env.SESSIONS_TABLE,
      Key: { sessionId: this.sessionId }
    };
    
    try {
      const result = await this.docClient.send(new GetCommand(params));
      return result.Item || null;
    } catch (error) {
      console.error('Error fetching session context:', error);
      throw new ContextStoreError(error.message);
    }
  }
  
  // セッションコンテキスト更新
  async updateSessionContext(updates) {
    const now = Date.now();
    
    const updateExpressions = [];
    const expressionAttributeNames = {};
    const expressionAttributeValues = {};
    
    for (const [key, value] of Object.entries(updates)) {
      updateExpressions.push(`#${key} = :${key}`);
      expressionAttributeNames[`#${key}`] = key;
      expressionAttributeValues[`:${key}`] = value;
    }
    
    // 更新時間も追加
    updateExpressions.push('#updatedAt = :updatedAt');
    expressionAttributeNames['#updatedAt'] = 'updatedAt';
    expressionAttributeValues[':updatedAt'] = now;
    
    const params = {
      TableName: process.env.SESSIONS_TABLE,
      Key: { sessionId: this.sessionId },
      UpdateExpression: `SET ${updateExpressions.join(', ')}`,
      ExpressionAttributeNames: expressionAttributeNames,
      ExpressionAttributeValues: expressionAttributeValues,
      ReturnValues: 'ALL_NEW'
    };
    
    try {
      const result = await this.docClient.send(new UpdateCommand(params));
      return result.Attributes;
    } catch (error) {
      console.error('Error updating session context:', error);
      throw new ContextStoreError(error.message);
    }
  }
  
  // 会話履歴追加
  async addToConversationHistory(message) {
    const now = Date.now();
    const messageId = uuid.v4();
    
    const params = {
      TableName: process.env.CONVERSATION_HISTORY_TABLE,
      Item: {
        sessionId: this.sessionId,
        timestamp: now,
        messageId: messageId,
        sender: message.sender,
        content: message.content,
        type: message.type || 'text',
        metadata: message.metadata || {}
      }
    };
    
    try {
      await this.docClient.send(new PutCommand(params));
      return messageId;
    } catch (error) {
      console.error('Error adding to conversation history:', error);
      throw new ContextStoreError(error.message);
    }
  }
  
  // 会話履歴取得
  async getConversationHistory(limit = 10) {
    const params = {
      TableName: process.env.CONVERSATION_HISTORY_TABLE,
      KeyConditionExpression: 'sessionId = :sessionId',
      ExpressionAttributeValues: {
        ':sessionId': this.sessionId
      },
      ScanIndexForward: false, // 降順（最新が先）
      Limit: limit
    };
    
    try {
      const result = await this.docClient.send(new QueryCommand(params));
      return result.Items.reverse(); // 古い順に並べ替え
    } catch (error) {
      console.error('Error fetching conversation history:', error);
      throw new ContextStoreError(error.message);
    }
  }
  
  // エージェント状態取得/設定
  async getAgentState(agentId) { /* 実装 */ }
  async updateAgentState(agentId, updates) { /* 実装 */ }
  
  // その他のユーティリティメソッド
}

class ContextStoreError extends Error {
  constructor(message) {
    super(message);
    this.name = 'ContextStoreError';
  }
}
```

```javascript
class AgentCommunicator {
  constructor() {
    this.sqs = new SQSClient({ region: process.env.AWS_REGION });
  }
  
  // メッセージ送信
  async sendMessage(targetAgentId, message) {
    const messageBody = JSON.stringify({
      messageId: uuid.v4(),
      timestamp: Date.now(),
      targetAgentId: targetAgentId,
      payload: message
    });
    
    const params = {
      QueueUrl: process.env.AGENT_QUEUE_URL,
      MessageBody: messageBody,
      MessageAttributes: {
        AgentId: {
          DataType: 'String',
          StringValue: targetAgentId
        }
      }
    };
    
    try {
      const result = await this.sqs.send(new SendMessageCommand(params));
      return result.MessageId;
    } catch (error) {
      console.error('Error sending message to agent:', error);
      throw new AgentCommunicationError(error.message);
    }
  }
  
  // タスク委譲
  async delegateTask(targetAgentId, taskData, context) {
    return this.sendMessage(targetAgentId, {
      type: 'TASK_DELEGATION',
      taskData: taskData,
      context: context
    });
  }
  
  // 処理結果通知
  async notifyResult(targetAgentId, resultData, originalTaskId) {
    return this.sendMessage(targetAgentId, {
      type: 'TASK_RESULT',
      resultData: resultData,
      originalTaskId: originalTaskId
    });
  }
}
```

### 8.2 SQSメッセージ処理

```javascript
// Lambda SQSハンドラー関数
exports.sqsMessageHandler = async (event) => {
  const agentController = new AgentController(
    getAgentConfig(),
    new ContextStore()
  );
  
  const responses = [];
  
  for (const record of event.Records) {
    try {
      // メッセージ解析
      const messageBody = JSON.parse(record.body);
      const targetAgentId = messageBody.targetAgentId;
      
      // 対象確認
      if (targetAgentId !== process.env.AGENT_ID) {
        console.warn(`Message for different agent received: ${targetAgentId}`);
        continue;
      }
      
      // メッセージ処理
      const response = await agentController.processAgentMessage(messageBody);
      responses.push({
        messageId: messageBody.messageId,
        response: response
      });
    } catch (error) {
      console.error('Error processing SQS message:', error);
      responses.push({
        messageId: record.messageId,
        error: error.message
      });
    }
  }
  
  return {
    statusCode: 200,
    body: JSON.stringify({ responses })
  };
};
```

### 8.3 Step Functions連携

複雑なエージェント連携フローは、AWS Step Functionsを使用して実装します：

```javascript
// Step Functions連携モジュール
class WorkflowOrchestrator {
  constructor() {
    this.sfn = new SFNClient({ region: process.env.AWS_REGION });
  }
  
  // ワークフロー開始
  async startWorkflow(workflowType, input) {
    // ワークフロータイプに基づくステートマシンARN選択
    const stateMachineArn = this.getStateMachineArn(workflowType);
    
    const params = {
      stateMachineArn: stateMachineArn,
      input: JSON.stringify(input),
      name: `${workflowType}-${Date.now()}`
    };
    
    try {
      const result = await this.sfn.send(new StartExecutionCommand(params));
      return result.executionArn;
    } catch (error) {
      console.error('Error starting workflow:', error);
      throw new WorkflowError(error.message);
    }
  }
  
  // ワークフロー状態確認
  async checkWorkflowStatus(executionArn) {
    const params = {
      executionArn: executionArn
    };
    
    try {
      const result = await this.sfn.send(new DescribeExecutionCommand(params));
      return {
        status: result.status,
        output: result.output ? JSON.parse(result.output) : null,
        startDate: result.startDate,
        stopDate: result.stopDate
      };
    } catch (error) {
      console.error('Error checking workflow status:', error);
      throw new WorkflowError(error.message);
    }
  }
  
  // ステートマシンARN取得
  getStateMachineArn(workflowType) {
    const workflowMap = {
      'relay': process.env.RELAY_WORKFLOW_ARN,
      'parallel': process.env.PARALLEL_WORKFLOW_ARN,
      'hierarchical': process.env.HIERARCHICAL_WORKFLOW_ARN
    };
    
    return workflowMap[workflowType] || process.env.DEFAULT_WORKFLOW_ARN;
  }
}

class WorkflowError extends Error {
  constructor(message) {
    super(message);
    this.name = 'WorkflowError';
  }
}
```

```javascript
// API Gateway Lambdaオーソライザー
exports.authorizer = async (event) => {
  try {
    // JWTトークン取得
    const token = event.authorizationToken.replace('Bearer ', '');
    
    // Cognitoによる検証
    const tokenVerifier = new CognitoJwtVerifier({
      userPoolId: process.env.USER_POOL_ID,
      clientId: process.env.CLIENT_ID,
      tokenUse: 'access'
    });
    
    const payload = await tokenVerifier.verify(token);
    
    // ポリシー生成
    return {
      principalId: payload.sub,
      policyDocument: {
        Version: '2012-10-17',
        Statement: [
          {
            Action: 'execute-api:Invoke',
            Effect: 'Allow',
            Resource: event.methodArn
          }
        ]
      },
      context: {
        userId: payload.sub,
        email: payload.email,
        groups: payload['cognito:groups']
      }
    };
  } catch (error) {
    console.error('Authentication error:', error);
    throw new Error('Unauthorized');
  }
};
```
