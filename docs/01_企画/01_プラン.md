# aiDev MVP開発プラン（改訂版）

## 🌟 プロジェクト目的

「専門業者に依頼していたクリエイティブ制作をAIとの対話だけで実現する」という未来を目指し、その第一歩として対話型AIを活用した「AI開発支援窓口（プリセールスエージェント）」を構築する。

SIer、Web制作会社、コンテンツ制作会社などに依頼していた専門的な制作プロセスを、AIがユーザーの意図を理解し自動化するプラットフォームを段階的に実現する。最初はAWS環境の設計・構築から始め、将来的には様々な業界の制作工程をAIが代行し、誰もが思い描いたものを簡単に形にできる世界を創造する。

---

## ✅ 到達目標（MVP）

- 自然言語による開発相談ができる対話型インターフェース
- 課題のヒアリングと解決提案が可能なAIエージェント
- ナレッジベースから情報を参照し、具体的かつ再現性のあるアウトプットを生成、および自動メンテナンス
- クラウド環境の自動設計・構築機能【環境構築自動化機能・環境発行機能】

---

## 📊 KPI・評価指標

- **営業効率指標**
  - 案件創出数：月間20件以上
  - 商談化率：AIチャット利用者の30%以上
  - AWS導入コンバージョン率：商談の40%以上
  - SES契約への展開率：AWS導入案件の25%以上

- **ユーザーエンゲージメント**
  - 平均セッション時間：15分以上
  - 再訪問率：70%以上
  - 提案満足度：4.0/5.0以上（フィードバック調査）

- **ビジネス指標**
  - 初期対応から契約までの期間：従来比30%短縮
  - 顧客獲得コスト(CAC)：従来比25%削減
  - 顧客生涯価値(LTV)：20%向上

## 📦 システム構成概要

```
🔼 対話型インターフェース
   └️ ユーザーインターフェース（Web/モバイル/メッセージングアプリ）
      ↓
📌 API層
      ↓
💡 AI処理エンジン
      - 入力処理 / 認証 / 制御
      ↓
📁 AIエージェント + ナレッジベース
      - ヒアリング / 設計支援 / 提案生成
      ↓
  【環境構築自動化機能】 クラウド環境の自動設計・構築
      - セキュアなアクセス制御
      - インフラストラクチャアズコード
      - クラウドリソース自動構築

  【環境発行機能】 新規クラウド環境発行 + 自動構築
      - アカウント管理
      - 自動初期構成
      - ガバナンス設定

  【運営者向け】 同じ機能を使って自社サービスを改善
      - 実践的運用
      - 開発テンプレ・プロンプトのチューニング
      - アップデート自動化
```

## ⛏ 主要機能の詳細

### 環境構築自動化機能
- AIと対話しながら、構成提案〜構築までを支援
- 安全なアクセス制御メカニズム
- コードとして定義されたインフラで自動化

### 環境発行機能
- クラウドアカウント管理による子アカウント発行
- ベストプラクティスに基づく初期セットアップ
- すぐに環境構築自動化機能を適用し構築開始可能

### 運営者向け活用
- 自社でも同じ機能を使って開発・改善
- ユーザー向けナレッジやテンプレートを随時反映
- 実運用を通じた信頼性強化にも貢献

- 判断基準：パートナー経由案件30%以上、ユーザービジネスへの貢献度測定
